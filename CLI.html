Alapbeállítások



Tűzfal kikapcsolás, címzés, szerver átnevezése:

Tűzfal kikapcsolása:
--------------------
Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled False

Ellenőrzés:
-----------
Get-NetFirewallProfile | Format-Table -Property Name,Enabled


Címzés:

netsh interface ipv4 show interfaces
-------------------------------------
Ki listázza a számítógépben található hálózati kártyákat
(célszerű először csak az internetest a gépbe helyezni)
ezesetben 1 lesz "Ethernet" néven
-----------------------------------------------------------
netsh interface ipv4 set address name="Ethernet" static cím maszk átjáró
------------------------------------------------------------------------
Beállítja a kiválasztott hálózatikártya ipv4-es címét
-----------------------------------------------------
netsh interface ipv4 add dnsserver name="Ethernet" 127.0.0.1
---------------------------------------------------------
Megadja a kiválasztott hálózati kártyának a DNS szerver címét
-------------------------------------------------------------


Számítógép átnevezése:

----------------------
wmic computersystem where caption='%computername%' rename új_gép_neve
---------------------------------------------------------------------
Átnevezi a számítógépet

a '%computername%' egy változó automatikusan a jelenlegi gépnevet illeszti be a helyére
---------------------------------------------------------------------------------------
